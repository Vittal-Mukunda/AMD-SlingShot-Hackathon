# Team Onboarding
Welcome to the RL-Driven Agentic Project Manager backend. The complete system simulates a dynamic work environment with continuous task arrivals and heterogeneous workers facing fatigue-induced efficiency drops, all managed by an Online Deep Q-Network.

This folder contains necessary documentation for understanding and tweaking the core functionalities of the simulation.

## Contents
1. [Tweaking Parameters](TWEAKING_PARAMETERS.md): How to adjust runtime and environment settings in `config.py`.
2. [Worker Profiles](WORKER_PROFILES.md): How to understand, add, and override worker traits.
3. [Mathematical Modeling](MATH_MODELING.md): Theoretical explanations of how our overload efficiency decay and reward balancing works.
4. [Task Generation](TASK_GENERATION.md): How dynamic tasks scale and how dependencies execute.
